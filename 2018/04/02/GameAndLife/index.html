网吧

发表于 2018-04-02 | 更新于: 2018-07-06 |
字数统计: 5.2k | 阅读时长 ≈ 0:05

已经不记得是什么时候就知道这世界上有网吧这个东西存在了，总之那个时候的我还很小，那个时候小镇上营业的网吧也还有好几家，那个时候的网吧也还叫网络会所。

每次路过，都能看见上面贴着当时最流行的游戏贴画，我还记得最多的就是魔兽世界跟cs的人物模型画，当然等我明白这些东西是什么的时候那些网吧都已经消失了。总之各种游戏人物贴的花里胡哨，怎么酷炫怎么来，然后再加上几个红字，xx网络会所，千兆光纤，空调包间。那时候还小，不知道什么叫光纤，就觉着前面能加上一个千字形容，很牛逼就是了。

再大些，去了一次网吧；还是班上的“大佬”们请客，去了网吧一脸懵逼，当时我对电脑的认识还停留在打字游戏上——因为当时学校有信息技术课，专门拿着金山2003练打字，一周上半节课。之所以是半节因为学校机房太小了，一次容不下一个班的人，只能分为两组，每组上半节课。不过当时也没有谁认真练打字的，不是在玩太空大战就是堆纸牌。连管我们上课的那个胖女老师整节课自己都在玩纸牌游戏。

第一次进了网吧算是打开了新世界的大门。当时单机游戏流行玩侠盗飞车，看着那些同学背各种秘籍比背课文来劲多了；联机就是cs反恐精英跟魔兽了。第一次打开cs 的时候不知道怎么进房间，问了旁边一位大哥，他说要输密码，然后密码是多少多少，然后进去之后却只能以第三人称视角看着其他人跑来跑去突突突。当时也不知道咋回事，再问那大哥他说再等等，等这一局完了就我就能玩了。结果傻傻等了十来分钟也没什么变化，一气之下就关了电脑看别人玩游戏去了。现在想想大概是以观察者身份进的游戏… …

当时的电脑还是那种阴极射线管的屏幕，记得当时摸屏幕的时候还被静电电了一下，当时可把我吓一跳，以为屏幕上带电… … 。倒是不记得那次是怎么结束的，不过算是我的网吧启蒙之旅吧。

真正开始混迹于各个大小黑网吧时已经是上初中的时候了，当时考上了市里面的一所私立初中，离开了上了六年小学的小镇。小城市就是有小城市的好，黑网吧随处可见；可能是当时上面抓的不严，网吧主要服务对象还是我们这些未成年的学生，从初中生到高中生，经常也能看见两三个小学生一块挤在一部电脑前。那时候学校举行升旗仪式星期一，一般都能看见几个因为翻墙外出上网被抓住而作检讨的学生。

当时上网要么就是办临时卡，费用贵上一两块钱一小时；要么就是花上二十块钱办上一张会员卡，但是需要身份证；因为当时还没有成年也没有身份证，一般都是开临时卡，这种时候就会很羡慕那些能从口袋里摸出一张身份证的高年级生，他们往往都是十分豪气的把身份证往前台一拍，喊上一嗓子，“老板，冲个会员”；那个场景在我这个还是初中小朋友的眼里，像极了那武侠电视剧中决战场景，大侠伸手拔出自己的神兵利刃，朝着反派大喊一句：“看招”；那动作，那气势，令我心驰神往。

从初二开始第一次去离学校的不远的网吧上了一次网之后，几乎之后的每次放半月假的回家与返校路上，都要找个地方待上一两小时，等到我上完网去车站等车的时候已经没啥人了，下车就都是下午三四点，我妈那个时候每次都是骑自行车来接我，就得多等上一两小时。现在想想挺不是滋味的。我记得初中最后一次家长会我就在台上反省自己，当着所有家长的面说以后再也不去上网了,当时还抱着我妈哭了。结果到后来还是没能改掉这毛病… …

再后来就考上了初中学校隔壁的高中。初中学校的机房从我们入学时就说已经建设，结果到初中毕业了连网线都没见过。所以高中的时候为了有机会去机房偷偷打游戏，就报名了当时的信息学奥赛培训，这个选择也就促使了我毕业时选择了大学里现在这个专业。仔细想想其实挺有意思的，我们永远都不知道现在的某些看起来很小的选择会对自己的未来产生有多大的影响，我人生的中几个重要的转折都是我几个不经意间的选择造成的结果。不过想想也就是充满了未知人生才有意思啊，就像每次去网吧之前都不知道这次会有什么样的游戏体验一样，总要充满着对未来未知的期待我们才会拥有继续生活下去的力量。

我人生中在网吧度过的第一个通宵是在高一，那时候正放寒假，我和几个同学留下来在学校弄竞赛培训，每天白天没事，晚上培训。某天晚上培训完外面正下大雨，几个人一合计反正晚上也没老师来查我们寝，就想着要不出去上个通宵的网。正愁怎么从门卫眼皮子底下混出去呢，刚好有个走读生的舅舅开车来接他，于是乎几个人就挤在一辆汽车的后座里偷偷出了校门。然后让他舅舅在学校方圆一公里内唯一的一个网吧停了车，冒着雨就冲了进去。现在已经很难再有当时的心境了，毕竟是第一次去网吧上通宵，对当时把学校看做是一座监狱的我们来说，那就是最高形式的自由。

不过到了后半夜才发现这自由的代价似乎挺严重，前半夜还是兴致冲冲打游戏的我们，后半夜就体会到了熬夜的难受之处，头昏脑涨眼睛疼，也就觉得这眼前的游戏似乎也并没有多大意思。特别是第二天清早回到学校， 因为宿舍门过了早上七点就会锁上铁门，我们就没法回去睡觉，一上午只好待在教室度过。南方的冬天总是难熬的，特别是那时教室还没有空调，所以那大概是那个冬天最难熬的一个上午了：几个人一晚上没睡觉困得不行，偏偏想睡觉又实在冻得睡不着，最后只好干坐着从早晨等到中午，等着宿舍开门，可以说是非常惨了。这段惨痛的经历也导致后来我整个高中生涯结束之前再也没有想去网吧上通宵的欲望了。

记得高中有次放假去网吧，碰到一个一边听歌，一边开着记事本看小说的奇葩同学。我一问才知道这货来网吧就为了看小说，游戏什么的也就是看小说看累了才打打，在当时可以说是非常的非主流了。反正我十分非常的不理解这种行为，我就觉得这人脑回路一定有问题。结果一交流人家还挺淡定，露出一副当你是没见过世面土包子的欠揍表情。不过一想到这货高中以来跟人说话似乎都是这种欠揍的表情，我也就释然了，然后心满意足的离开了网吧，一脸发现世界上还有这种新兴人类的满足。

其实从初中到高中在老师同学们的眼里我也算个好好学生，不过当时社会主流舆论都把上网打游戏这一类的事情都归为坏学生的行为，所以我每次都是怀着十分内疚的心情去上网，进网吧之前出网吧之后都要进行一次深刻的自我反省；每次从网吧出来就想着下次一定不去了，结果反省了这么多年，路过网吧门口还是控制不住自己的脚步。

知道我上网打游戏的人也不多，一是当时也没什么人陪我一块去，二来也是下意识的不和别人谈论，毕竟好学生的形象还是要维持一下的。有次印象特别深刻的就是放假去了一家常去的网吧结果碰到了班上的纪律委员，是个妹子，当时我正愁来晚了没机位了咋办，结果她一眼就看见了我，然后用全网吧都能听到的声音喊了一句：“班长，原来你也来上网啊… … “,然后那一瞬间我就感觉我收到了不下十双眼睛行的注目礼；我第一反应就是：我给班长这个职业丢脸了… … 向来有尴尬癌的我当时就想从二楼窗户跳下去（我都不知道我怎么当上班长的）。我都不好意思直视她，也不知道怎么聊，就问她咋来了，她说等她哥，说完她哥就从座位上起身，似乎是要走了。然后我可爱的纪律委员就一口一个班长，很热情的表示，她哥就要走了，我可以去他那个位置上机。当时我都不好意思看他哥的表情，全程尬聊。

在市里上了六年学，去了五年的网吧，不敢说知道市里所有的网吧在哪儿，但前前后后去过的黑网吧怎么着也不止一双手的数目，从学校到车站的路上哪条小巷子里有家不要用身份证的黑网吧，哪家网吧网速快点，哪家网吧座位很烂，哪家的网吧最近不接无证游民，又有哪家网吧最近换新机子了。甚至还见证了几家网吧的兴衰，从周五放假期间的满座盛况到三三两两的上网人群再到换了新的网吧老板到最后网吧倒闭，整层出租。特别是有个网吧老板带着我二十多块大洋的网费跟着小姨子跑了，网吧说关门就关门，当时气得我啊。

有段时间听说有个网吧出了命案，说是社会人员冲突捅死了人，然后那段时间放假上网都提心吊胆的，生怕哪天因为坐了一个不该坐的位置就不小心被人盯上了，玩个游戏期间都东张西望的，跟做贼一样。

再后来就高三毕业了，高考完那个暑假在我堂姐工作的酒店里找了一个行李员的暑期工，每天也就站站岗，帮别人存存行李，每天工作八小时，剩下时间也没啥事干。本来挺无聊的，有次聊天听师傅说八楼有个黑网吧，当时就来劲了。下了班换了衣服就坐电梯直达八楼，果然，黑网吧就是黑网吧，连窗户都拉上了窗帘，大白天跟阴曹地府似的，只有屏幕前的一张张脸反射着幽幽绿光。同在八楼的好像是个洗脚店，他们的前台就正对着电梯，到后来我都怀疑那些上夜班的前台小姐姐们都认识我了，因为我每次都是玩到晚上十点才走，还一脸因为游戏打多了的肾虚样。

网吧很小，就是居民间改造的，还有一个小厨房，饭点能点盒饭，十几块钱一份。老板是对中年夫妇。整个网吧除了厨房就分了两个房间，一个房间摆了一个收费结账的前台和几张沙发，晚上老板就睡在沙发上。隔壁就是摆电脑的地儿，各种奇奇怪怪的线胡乱的在墙上交织着，不过十来台电脑，座椅还是被磨掉许多皮看上去用了五六年的的沙发座椅。办了个会员，冲五十送五十，冲一百送一百，看上去挺实惠的，对吧。一个小时五块钱！！到现在我学大学旁的网吧也没这么贵。五块一小时没事，就一次冲上五十一百的，就相当于两块五了，我当时觉得我真是机智。然后我就开始疯狂打游戏，暑假一共干了差不多一个半月多一点，反正最后除掉没有修的假期，给我发了两个月工资。然后差不多两个月有一个多月每天除了吃饭睡觉加八小时上班之外就待在那里，前前后后冲了四次还是五次费，前两次都是二十一冲，后来就是五十一百的。加上他送的网费也就是说一共是花了四五百，换成时间也就是八九十个小时，每天都是起码三个小时以上。也就是那时候通过打游戏认识了很多朋友，虽然现在还联系的就剩下一个了，但是那段上网的时间大概是我上学以来所有上网经历中最轻松最开心的时候了。

再后来就上大学了，大一的时候没去过网吧，一是因为当时的我还没有成年，也不知道这边网吧正不正规 ，让不让我这种小朋友进去，再就是自己也有电脑了，也没什么必要出去。直到某天跟老朋友们开黑时被校园网气得差点把电脑砸了，突然想起还有网吧这种东西。

大二的时候刚好成年，有了去网吧底气。第一次能把身份证往前台一拍，然后十分豪气的喊出那句羡慕了别人很多年的——“老板，充个会员”，那种感觉就像是一个偷偷摸摸在社会上摸爬滚打了四五年的小贼，突然在某一天有了一个正大光明的身份，整个人的精神气都不一样。

偶尔放长假回家去市里跟老友聚完会就会在以前偷偷摸摸上网的老地方一个人正大光明的打上一个通宵。不用担心会因为遇到认识的人而尴尬，也不用担心睡完起来会因为没有成年而被大清早赶出门，只是总有一种怅然若失的感觉萦绕心头。

时代发展，当年偷偷摸摸去的很多黑网吧不是倒闭就是改革换代换了门面换了老板换了设备，清一色大白框显示器，看不到各种杂乱电线的整洁桌面，以及已经改叫网咖而不叫网吧的名字。以前去网吧都是坐上两三小时就走，或是因为头疼头晕不想玩又或者是实在赶时间了又或是其他原因。现在一进，越是一个人，就坐的越久，有时候一坐就是一个通宵，第二天清早吃过油条豆腐脑溜回宿舍睡觉。也不是因为游戏多好玩，就是单纯的想一个人待着，暂时逃避一下现实世界的纷扰。一个人去海边感觉挺无聊还得走来走去，去网吧就不一样了，戴上耳机，这个世界就与你无关，不管之前发生过什么事情，没有打开一个游戏解决不了的事。

有一件很奇怪的事就是，小学的时候听某位堂哥聊天说网吧通宵就只要十块，到后来初中听同学说也是十块，再到高中自己第一次通宵还是十块。上大学刚来这里的时候我记得也是一个通宵十块，前段时间去了一趟终于涨到十五了。要是房价能有这种趋势就好了。

常年混迹于小城市黑网吧，对我来说，五块钱一个小时的价格就已经是我能接受的价格上限，直到上个月跟同学去北京浪了一圈，带我这个乡下人见识了一下大城市中的高档网吧——普通区就已经是六块钱一个小时，包间的价格更是高达十块以上，就这一点就让我认识到了什么叫大城市与小城市是之间的差距。

转眼大学四年就要过完了，大二大三网吧去得比较频繁，大四上学期由于忙着考研，也就没时间再去，想想校外的那个网吧里还有没有用完的网费，这两天就念叨着是不是什么时候找个时间把网费上完了，奈何当年一块开黑的小伙伴有两个今年准备考研，剩下两个也约不起来，估摸着又只能自己一个人去了。

不知道从什么时候开始，网吧对我而言已经不再是一个只为娱乐而去的地方，更多的时候把它当做一个逃避现实的港口，不在乎一个人玩游戏好不好玩，只是享受那几块钱一个小时廉价却又难得的时光，没有作业，没有理想，没有烦恼，只有一个泛着蓝光的屏幕、一副廉价耳机和一个沾满烟灰的键盘，这些东西就能组成一个可以让我暂时忘记一切的虚拟世界。

年少的我有些贪玩，愿长大后的我还能保持一颗童心，不会被现实磨掉棱角，梦想还是最初的模样，还会相信爱情的纯真。

敬那些年，在网吧度过的青春。
